TV NUSANTARA 45 — Static News Site (Free) - Netlify + GitHub

Isi:
- index.html, style.css — frontend
- /articles — sample article pages
- /content/articles.json — data file used by index
- /admin — Netlify CMS admin (requires Netlify & Git Gateway)
- /assets/logo.png — logo yang kamu kirimkan
- tentang.html, kontak.html — halaman tambahan

Langkah cepat deploy ke Netlify:
1. Buat repository GitHub (misal: tvnusantara45) dan push seluruh isi folder ini.
2. Di Netlify: New site from Git -> Hubungkan GitHub -> pilih repo -> Deploy.
3. Aktifkan Identity (Site settings -> Identity -> Enable) dan Git Gateway.
4. Buka https://<sitename>.netlify.app/admin/ untuk masuk ke Netlify CMS.
5. Sesuaikan email dan domain di README, jika perlu.
